A sleek, minimalist Linux desktop configuration based on the i3 window manager, featuring a dark charcoal grey base with vibrant candy apple red accents. Designed for clarity, focus, and aesthetic precision—perfect for terminal work, coding, and ricing enthusiasts.

## Installation
> [!IMPORTANT]
> Keep in mind, this script is made for Devuan GNU/Linux and its child distributions: your milage may vary.

1. On GitHub, either:
  - Select **Code > Download ZIP**.

    OR

  - ```bash
    git clone https://github.com/wolfdaemon/dotfiles-rice-charcoal-candy-apple.git
    ```
2. Navigate to the directory of the project downloaded:
```bash
cd dotfiles-rice-charcoal-candy-apple
```
3. Make the installation script executable:
```bash
sudo chmod +x install.sh
```
4. Execute the install script:
```bash
./install.sh
```
5. Follow any additional necessary instructions printed out by the script.
